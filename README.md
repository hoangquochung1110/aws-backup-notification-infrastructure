# A comprehensive Infrastructure as Code (IaC) solution demonstrating automated backup management on AWS, featuring:

## Core Components
**CloudFormation** templates for infrastructure provisioning

**AWS Backup** for automated backup management

**SNS-based** notification system

**Lambda**-powered backup testing

## Lab overview

1. Deploy the Infrastructure
2. Create Backup Plan
3. Enable Notifications
4. Test Restore
5. Teardown

### Step 1:
- Create a S3 bucket and upload `backup-lab.yaml` and `lambda_function.zip`.
